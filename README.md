# 🌐 API Testing Project

## 📌 Descripción
Proyecto de pruebas API enfocado en la validación de endpoints REST mediante solicitudes HTTP.

---

## 🎯 Objetivo
Verificar el correcto funcionamiento de endpoints y validar respuestas del servidor.

---

## ✅ Actividades realizadas
- Envío de solicitudes POST
- Validación de respuestas HTTP
- Comparación entre resultado esperado y obtenido
- Documentación de pruebas

---

## 🛠️ Herramientas
- Postman
- API Testing
- JSON Validation

---

## 🔎 Pruebas realizadas
- Endpoint validation
- Response status verification
- Functional API testing
